김민준을 소개합니다.

대학 졸업을 앞둔 컴퓨터 공학 전공의 학생, 김민준입니다. 전공 수업과 여러 프로젝트를 통해 소프트웨어 개발에 필요한 다양한 프로그래밍 언어와 기술을 익혔습니다. 특히, 인공 지능과 모바일 애플리케이션 개발에 깊은 관심을 가지고 있습니다. 대학교에서는 동아리 활동을 통해 팀워크와 리더십 경험도 쌓았습니다. 졸업 후에는 이러한 기술과 경험을 바탕으로 IT 분야에서 전문 개발자로 성장하고자 합니다.

Front-End

웹 개발에 관심이 많아, 프론트엔드 개발에 필요한 여러 기술을 스스로 학습하고 있습니다. 제가 주로 사용하는 프론트엔드 기술은 HTML, CSS, JavaScript입니다. 이를 기반으로 최근에는 React.js 프레임워크를 배우기 시작했습니다​.
React를 통해 컴포넌트 기반의 개발 방식과 상태 관리의 중요성을 이해하게 되었고, 이를 실제 프로젝트에 적용해 보았습니다. 또한, Redux를 사용하여 복잡한 상태 관리를 경험했으며, 이 과정에서 불변성을 유지하는 방법과 효율적인 데이터 흐름을 설계하는 방법을 배웠습니다.
프로젝트를 진행하면서 CSS 프레임워크인 Bootstrap과 Material-UI를 사용해 보았습니다. 이 도구들은 반응형 디자인을 손쉽게 구현할 수 있게 해주어 사용자 경험을 크게 향상시키는 데 기여했습니다. 이러한 경험을 통해 사용자 중심의 UI/UX 디자인의 중요성을 더 깊이 이해하게 되었습니다.

Back-End

백엔드 개발에 특히 관심이 많아, 이 분야에 필요한 여러 기술을 깊이 있게 학습하고 있습니다. 제가 주로 사용하는 백엔드 기술은 Python과 Java입니다. 이 언어들을 사용하여 서버 사이드 로직과 데이터베이스 관리를 합니다.
Node.js와 Express 프레임워크를 통해 비동기 처리와 미들웨어의 활용 방법을 배우며, RESTful API 설계 방법을 실습했습니다. 또한, MongoDB와 같은 NoSQL 데이터베이스를 사용하여 대용량 데이터 처리와 비구조적 데이터의 효율적 관리 방법을 경험했습니다.
데이터베이스 관리 시스템으로는 MySQL과 PostgreSQL을 사용해보았고, 이를 통해 SQL 쿼리 최적화와 트랜잭션 관리 기술을 개발했습니다. Docker와 같은 컨테이너 기술을 사용하여 어플리케이션의 배포 및 관리를 더욱 효율적으로 할 수 있게 되었습니다.
이러한 기술들을 통해 서버의 성능을 최적화하고 보안을 강화하는 방법을 배웠으며, 클라우드 서비스(AWS, Azure)를 이용한 환경 구축에도 참여했습니다. 앞으로도 체계적인 백엔드 시스템 구축과 성능 최적화를 위해 지속적으로 학습하고, 전문 백엔드 개발자로 성장하겠습니다.

경력

  • 학력: 한국대학교에서 철학과를 졸업.
  • 교육과정: 한국폴리텍대학의 데이터분석과 과정을 수료하였습니다.
  • 수상 경력: 구글에서 주최한 해커톤에서 최우수상을 수상하였습니다.
  • 로또 프로그램 개발(클릭)
